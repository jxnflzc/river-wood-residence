    Mkdocs搭建¶

    Mkdocs安装¶

    1. 安装Python

    2. pip 安装
      pip install mkdocs

    3. 检查是否安装成功
      mkdocs --version

    Mkdocs使用¶

    1. 工程创建
      mkdocs new <my-project>
    2. 进入工程目录
      cd my-project
    3. 启动服务器
      mkdocs serve
    4. 浏览器中输入网址
      http://127.0.0.1:8000/
    5. 新建页面
      在<my-project>/docs下新建md文件，并编辑好内容。（参考Markdown基本语法)
    6. 配置页面
      在mkdocs.yml文件中配置新增的页面。（参考mkdocs.yml范例)
